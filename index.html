FULL-COUNT

Every pitch. Every play. Every out.

About
full-count is a keyboard-driven TUI for scoring baseball games in real time. It is written in Rust and built for fast, no-mouse scorekeeping.

Quick Start 
cargo build --release

./target/release/full-count
./target/release/full-count --load cubs-vs-sox
./target/release/full-count --load cubs-vs-sox --replay

Main Keys
  • B ball, S strike, F foul
  • 1/2/3 hit types, H home run
  • K strikeout swinging, L strikeout looking
  • W walk, P hit by pitch, C fielder's choice
  • G groundout, D double play, O flyout, E error, V sac fly
  • F2 save, U undo, X end game, Q quit
